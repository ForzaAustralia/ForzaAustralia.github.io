Instant-runoff voting tool

A Website that helps you calculate the result of an Instant-runoff vote.

About

This is a website that helps you calculate the result of an instant-runoff vote. It's programmed with javascript powered by jQuery and structured and styled with HTML and CSS on the basis of bootstrap. The Sourcecode is licensed under the MIT License and available on GitHub. It was written by Peter Grassberger aka. PeterTheOne.

Input

Candidates

Please enter the names of the candidates, one per line.

Example:

Barack Obama
Mitt Romney
...

Ballots

Please enter one ballot per line. A ballot consists of ranking number 1 2 3 4 ... in a row separated by a comma,. A ballot has the same amount of ranking numbers as there are candidates. The first number is the rank of the first candidate on this ballot, etc. Spaces are ignored.

Example:

3, 2, 1, 4
4, 3, 1, 2
1, 3, 4, 2
...
Accept incomplete ballots
