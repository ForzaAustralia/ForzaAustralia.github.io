Input

Candidates

Please enter the names of the candidates, one per line.

Example:

Barack Obama
Mitt Romney
...

Ballots

Please enter one ballot per line. A ballot consists of ranking number 1 2 3 4 ... in a row separated by a comma,. One ballot has the same amount of ranking number as there candidates. The first number is the rank of the first candidate on this ballot, etc. Spaces are ignored.

Example:

3, 2, 1, 4
4, 3, 1, 2
1, 3, 4, 2
...
